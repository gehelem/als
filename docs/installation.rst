############
Installation
############

The best way for now is to get the latest nightly build from ALS website.
Browse to https://als-app.org/nightlies/latest and pick the package that suits your target system.

Widows
======

Download the .exe file and simply double click it :)

Linux for desktop PCs
=====================

Download the .run file, make it executable and then launch it :)

Linux for 32bit Raspberry Pi
============================

Download the .tgz file wherever you wish, then execute the sdfsd file in there :)

macOS
=====

Download the .dmg file and let the apple do its magic :)
